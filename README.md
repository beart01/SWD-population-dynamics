# SWD-population-dynamics
This is a Leslis Matrix- style simulation for population dynamics of spotted-wing drosophila (SWD) (Drosophila suzukii; Diptera: Drosophilidae). It uses temperature and host availibility data (crop and non-crop hosts) to drive a time-step simulation. Mortality and fecundity of SWD are driven by physiological age (Wiman et al. 204).
